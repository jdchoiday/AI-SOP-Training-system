# SOP Source (Drive → 기초 텍스트)

이 폴더는 Google Drive의 SOP 라이브러리
(`https://drive.google.com/drive/folders/1QWHnexl7O3CpqZlBlMx1tUZM2a2XWEs7`)를
**원문 그대로(요약·생략 없이)** 기초 텍스트(Markdown)로 변환해 모아둔 곳입니다.
이 텍스트는 SOP 훈련 시스템에서 SOP를 생성하는 1차 자료로 사용됩니다.

> ⚠️ 기존 저장소 파일은 전혀 수정하지 않습니다. 모든 변환 결과는 이 새 폴더(`sop-source/`)에만 들어갑니다.

## 변환 원칙
- **요약·축약·생략 금지.** 각 문서의 전체 내용을 충실히 전사합니다.
- **원문 언어 보존**(대부분 베트남어/일부 영어). 번역하지 않습니다.
- 제목·섹션 순서·번호 단계·표·주의사항 등 **구조 보존**(표는 Markdown 표로).
- 영상(.mov/.mp4)·단독 이미지는 변환 대상에서 제외(인벤토리에 링크만 기록).
- 각 파일 상단에 출처 메타데이터(front-matter) 포함.

## 폴더 구조 (Drive 카테고리 미러링)
```
sop-source/
  README.md                 ← (이 파일)
  _root/                    ← 루트에 있던 단일 문서들

```
