| :-: | :-: | :-: | :-: | :-: |
| 1 | Tháo tấm ván ép bộc nệm cũ và vệ sinh lại bề mặt keo cũ và bề mặt sàn |  |  | Mang bao tay xử lý lại tấm ván cũ |
| 2 | - Trải simili ra sàn (mặt trái hướng lên) và ván ép + mút vào giữa tấm simili - Bắt đầu từ 1 cạnh dài: kéo simili căng rồi dùng súng bắn ghim cố định vào mặt sau ván. Làm tương tự với cạnh đối diện, luôn kéo căng để mặt trên phẳng đẹp. |  |  | Kéo căng để mặt trên phẳng căng đẹp |
| 3 | Xử lý 2 cạnh ngắn  Gấp simili lại, kéo căng, bắn ghim cố định.  Dùng kẹp giữ tạm để dễ căn chỉnh (như trong hình đang làm). |  |  |  |
| 4 | Xử lý góc (quan trọng)  Ở góc, gấp simili thành nhiều nếp nhỏ (giống gói quà) để tránh bị phồng.  Bắn ghim giữ chặt các nếp. |  |  |  |
| 5 | Hoàn thiện  Kiểm tra toàn bộ các cạnh, thêm ghim nếu chỗ nào còn lỏng.  Cắt bỏ phần simili thừa.  Mặt trên phải căng, không nhăn. |  |  |  |
| 6 | Dùng keo silicone AB để dán tấm ván mới vào bề mặt sàn |  |  |  |
