INTERACTIVE ROOM

1.  OPENING SHIFT

1.  Nhân viên cần phải kiểm tra các thiết bị trước khi mở bao gồm:

1.  Máy lạnh
2.  Máy scan
3.  Remote điều khiển máy chiếu
4.  Bàn phím
5.  CPU
6.  Quạt tản nhiệt

2.  Mở tất cả các thiết bị bao gồm:

1.  Máy lạnh (25℃)
2.  Máy scan (nhấn giữ button 5s)
3.  Remote điều khiển máy chiếu (bấm nút xanh dương)
4.  CPU (bấm nút mở)
5.  Quạt tản nhiệt (bấm nút mở ở mặt lưng)

3.  Kiểm tra giấy vẽ (clean giấy cũ, refill giấy mới)

4.  Kiểm tra các hộp đựng bút màu, nếu có hư hỏng thì phải thay thế.

5.  Sắp xếp bàn, ghế ngay ngắn

2.  CLOSING SHIFT

1.  Sắp xếp bàn, ghế ngay ngắn

2.  Tắt tất cả thiết bị bao gồm:

1.  Máy lạnh
2.  Máy scan (nhấn giữ button 5s)
3.  Remote điều khiển máy chiếu (bấm nút xanh dương 1 lần để hiện thông báo tắt, bấm lần 2 để xác nhận tắt)
4.  CPU (sử dụng tổ hợp Ctrl + Esc để hiện menu window và tắt)

3.  Quạt tản nhiệt (bấm nút sau lưng)
