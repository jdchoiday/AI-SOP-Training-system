# 영상 코스 제작 계획 (Video Course Plan)

`sop-source/`의 227개 원문 텍스트를 검토해 **"영상 코스로 만들기 좋은 것"만** 큐레이션한 계획입니다.
기준: **서술형 절차/매뉴얼/가이드** = 영상 적합 ✅ / **양식·체크리스트·재고·명단·색인** = 참고문서(영상 부적합) ⚠️.

기존 라이브 7개와 동일 포맷으로 제작: **영상 대본(여러 씬, 베트남어 나레이션 + 디자인 슬라이드) + 퀴즈 4문항 + 다국어 본문**.
제작분은 `status=draft`(직원 비노출)로 올린 뒤 검토 → 발행. id 접두사 `kwzsrc-`.

---

## 요약

| 구분 | 개수(큐레이션) | 처리 |
|---|---|---|
| **Tier 1 — 핵심 운영 SOP** | ~27 | 개별 영상 코스 ✅ (최우선) |
| **Tier 2 — 놀이/게임 가이드** | 31 (또는 묶음 ~10) | 개별 짧은 영상 또는 테마 묶음 ✅ |
| **Tier 3 — 교육(Strong) 매뉴얼** | ~12 | 개별 영상 ✅ |
| **Tier 4 — F&B 서비스 절차** | ~7 | 개별 영상 ✅ |
| **통합 — 놀이존 Areas** | ~3 | 24개 얇은 체크리스트 → 소수로 **통합** ✅ |
| **제외 — 참고/스킵** | ~70 | 양식·명단(PII)·재고·색인 → 영상 안 만듦 |
| **이미 라이브** | 7 | (BEO·ELIS·Market·Workshop캔들/수업·Cashier데이터·오픈시프트) |

**영상 코스 목표: 약 60~80개** (게임을 개별로 하면 ~80, 묶으면 ~58). **완료: 1개** (미술수업, draft).

## Tier 1 — 핵심 운영 SOP (최우선, ~27)

### WORKSHOP (3 신규) — 캔들/수업은 라이브
- [x] 미술수업 (Art Class) — **제작완료(draft)** `kwzsrc-workshop-art-class`
- [ ] 슬라임 파티 셋업 (Slime setup)
- [ ] 워크숍 운영 절차 ×2 (SOP - QUY TRÌNH VẬN HÀNH WORKSHOP)

### CASHIER / INFORMATION 절차 (~15)
체크인·응대 / 인포카운터 오픈 / 인포카운터 마감 / 2025 결제 절차 / Cukcuk 오더 / VAT 세금계산서 / 파티 예약금·차감 / Cukcuk 예약금 생성 / 매출보고서 / Hostess 절차 / Thiso 전원 ON·OFF / 마감 사진 보고 / 지점 체크인-아웃 / 캐셔 정리·청소 / 온라인 시스템 등록
> 제외(양식성): Lost&Found 등록부

### PLAYGROUND Staff Manual (~9 고유, 영/베 쌍 통합)
놀이터 소개·목적 / 공통 규칙(General Rules) / 아이와 노는 법 / 아이 감독법 / 아이와의 기본 대화·소통 / 말투·목소리 톤 / 핵심 역량 / 하지 말아야 할 것(Don't) / 사고·비상 대응 절차

## Tier 2 — 놀이/게임 가이드 (31, 선택: 개별 또는 묶음)
Games Manual 31종(예: 줄넘기 KWZ-01 …) — 게임별 규칙·연령·인원·안전이 담긴 독립 가이드.
- **옵션 A**: 게임별 짧은 영상 31개(직원 현장 활용도 높음)
- **옵션 B**: 연령/유형별 테마 묶음 ~8~10개
+ Defect Manual 내 실제 게임/활동 시트 ~6개 추가.

## Tier 3 — 교육 프로그램(KWZ Strong) (~12)
Strong 교사 매뉴얼(교사 역량·태도·목소리 통제·do/don't·사고 대응·프로그램 스토리 등, 영/베 쌍 통합) + 지도용 운동/커리큘럼 중 서술형.
> 제외: 장비 재고표·월별 커리큘럼 표·출석/명단(데이터).

## Tier 4 — F&B 서비스 절차 (~7)
4 rules / 인사응대(Greeting) / 메뉴 안내 / 오더 받기 / 추가판매(Suggestive) / 트레이 서빙 / POS 사용 / 취소 절차 / 팀워크 중 서술형 절차.
> 제외: Monthly Training Plan(양식).

## 통합 — 놀이존 Areas Manual (24 → ~3)
D2/D7 존별 open/operate/close 체크리스트(평균 1KB, 반복적) → 통합 코스로:
1. 놀이존 오픈/운영/마감 표준 절차
2. 놀이존 안전·기구 점검(바닥·LED·탄성 등)
3. (필요 시 D2/D7 구분)

---

## 제외 — 참고문서/스킵 (~70)
- **CASHIER D2-FORM / D7-FORM (26)** — 업무 인수인계 등 **양식** → (옵션) 본문만 등록 또는 스킵
- **CLASS-EDU Camp (24)** — 캠프 **등록명단·출석부 = 고객 개인정보(PII)** → ⚠️ 교육영상 **제외**(민감정보)
- **재고표·커리큘럼 표·템플릿·색인** — 데이터/양식
- **영/베 중복본의 한쪽** — 통합 코스로 커버

---

## 진행 방식
1. Tier 1부터 배치 제작(미술수업과 동일 품질) → `draft`로 적재
2. 배치마다 관리자 검토 → 일괄 발행
3. Tier 2~4, 통합 순으로 확대
4. 제외군은 영상 안 만들고, 필요 시 참고문서로만 별도 처리

**현재 상태: 1/~70 제작완료 (미술수업, draft)**
