# SOP Training 베타 테스터 가이드

안녕하세요! SOP Training 시스템 소프트런칭 베타에 참여해 주셔서 감사합니다.
이 문서는 **약 10분**이면 읽고 바로 시작할 수 있도록 정리했습니다.

---

## 1. 시스템은 어떤 서비스인가요?

> 매장/직무별 업무 매뉴얼(SOP)을 **AI 가 영상 강의로 자동 생성**해서, 직원들이 게임처럼 학습할 수 있게 해주는 교육 플랫폼입니다.

- 👩‍🏫 AI 내레이션 + 자동 삽화 + 퀴즈로 구성된 "영상 챕터"
- 🎮 XP / 코인 / 스토어 교환 / 팀 미션 등 게이미피케이션
- 📊 관리자 대시보드: 진행률, 팀별 비교, 리포트

---

## 2. 접속 방법

### 🌐 URL
- **운영 서버**: https://your-production-url.vercel.app  *(관리자로부터 안내받은 주소)*
- **관리자 화면**: `/admin/`

### 🔑 로그인 계정
관리자가 **개별 안내**해드립니다. (이메일 + 임시 비밀번호)

> ⚠️ 최초 로그인 후 **반드시 비밀번호 변경**해주세요.

---

## 3. 먼저 해볼 것 (5분 체크리스트)

### ✅ STEP 1 — 로그인 / 프로필 확인
1. 안내받은 이메일 / 비밀번호로 로그인
2. `프로필` 메뉴에서 **이름·지점·팀** 이 맞게 표시되는지 확인
3. 잘못되어 있으면 → 👉 피드백 버튼(우측 하단 💬)으로 알려주세요

### ✅ STEP 2 — 학습 챕터 1개 끝까지 돌려보기
1. 홈에서 **배정된 SOP** 하나를 선택
2. 영상이 **처음부터 끝까지 끊김 없이 재생**되는지
3. **AI 내레이션 목소리**가 자연스러운지 (끊김/중간에 목소리 바뀜 체크)
4. 퀴즈가 정상 제출되고 **XP 가 오르는지**

### ✅ STEP 3 — 모바일에서도 테스트
- 휴대폰 브라우저 / 태블릿으로 같은 링크 접속
- 영상이 잘 나오는지, 피드백 버튼이 가려지지 않는지

### ✅ STEP 4 — 피드백 보내기
- 우측 하단 💬 버튼 → 아무 코멘트 1건 남기기
- (버그가 아니어도 OK — "이 폰트 너무 작아요" 같은 것도 환영)

---

## 4. 피드백 보내는 법

화면 **우측 하단 💬 버튼**을 누르면 모달이 뜹니다.

### 종류 5가지
| 아이콘 | 종류 | 언제 선택 |
|---|---|---|
| 🐛 | 버그 | 작동이 안 되거나 오류가 떴을 때 |
| 🎨 | UX | 사용하기 불편하거나 헷갈릴 때 |
| 📝 | 내용 | SOP 내용이 틀렸거나 부족할 때 |
| 💡 | 제안 | 있으면 좋겠다 싶은 기능 |
| 💬 | 기타 | 위 네 가지에 안 맞는 의견 |

### 심각도 4단계
- **가벼움** — 불편하지만 계속 쓸 수 있음
- **보통** — 불편해서 돌아가는 방법을 찾게 됨 (기본값)
- **심각** — 해당 기능이 사실상 사용 불가
- **치명적** — 데이터 손실/로그인 불가/완전 멈춤 같은 경우

### 잘 쓰는 피드백 예시

❌ 나쁜 예:
> "안 돼요"

✅ 좋은 예:
> "Chrome 모바일(아이폰 15)에서 7챕터 퀴즈 문제 3번 선택지를 눌러도 반응이 없어요.
> 선택이 안 되니까 제출 버튼도 회색으로 남아있고, 새로고침해도 같은 증상입니다.
> 2025-04-23 오후 3시쯤 처음 발견했어요."

👉 **어떤 상황 / 어떤 기기 / 어떤 동작 / 기대 결과** 4가지가 들어가면 최고예요.

---

## 5. 테스트해주시면 특히 좋은 부분

소프트런칭 기간에 **집중 검증이 필요한 영역**입니다:

1. **영상 재생 안정성** — 긴 챕터 (10분 이상) 완주 가능한지
2. **AI 목소리** — 씬이 바뀌어도 목소리가 일관된지
3. **퀴즈 채점** — 맞았는데 틀렸다고 나오는지
4. **XP / 코인 계산** — 예상한 만큼 들어왔는지
5. **모바일 레이아웃** — 버튼/텍스트가 잘려 보이는지
6. **로그인 / 비밀번호 변경** — 흐름이 자연스러운지
7. **이미지 생성 품질** — SOP 내용과 맞는 이미지가 나오는지

## 6. 자주 묻는 질문

<details>
<summary><b>Q. 영상이 중간에 멈춰요</b></summary>

- 새로고침(F5) 후 재생하면 이어집니다 (진도는 저장됨)
- 반복되면 어떤 챕터의 어느 위치인지 💬 피드백으로 알려주세요
- 네트워크가 불안정한 경우에도 발생할 수 있습니다
</details>

<details>
<summary><b>Q. 퀴즈가 틀렸는데 해설이 안 나와요</b></summary>

- 해설은 제출 직후 자동 노출됩니다. 안 나오면 버그입니다 → 💬
</details>

<details>
<summary><b>Q. 비밀번호를 잊어버렸어요</b></summary>

- 관리자에게 비밀번호 초기화를 요청해주세요
- (소프트런칭 기간은 self-service 비밀번호 재설정이 아직 없습니다)
</details>

<details>
<summary><b>Q. 내 진도가 갑자기 리셋됐어요</b></summary>

- ⚠️ 이건 심각한 버그입니다. **심각도: 치명적** 으로 즉시 피드백 해주세요
- 가능하면 스크린샷도 함께 부탁드립니다
</details>

<details>
<summary><b>Q. 친구한테 링크 공유해도 되나요?</b></summary>

- 🚫 소프트런칭 기간에는 **지정된 베타 테스터만** 접근해주세요
- 정식 오픈 전까지 외부 공유는 삼가주세요
</details>
