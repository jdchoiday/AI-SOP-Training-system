# 교재 프로젝트 폴더 가이드

## 시스템 구조 (중요!)

이 시스템은 **지점 + 팀 2단계 구조**로 직원/교재를 관리합니다.

```
지점 (실제 매장 위치)
  └── 팀 (사업 단위 = 브랜드)
      └── 직원
```

### 실제 예시

```
베트남 롱쑤웬점
├── AION 킨더 팀 (유치원 사업)       → 유치원 교재만 봄
├── 키우자 타운 팀 (키즈카페 사업)   → 키즈카페 교재만 봄
└── BBQ 로맨스 팀 (BBQ 사업)         → BBQ 교재만 봄

베트남 하이퐁점
├── AION 킨더 팀                    → 유치원 교재만 봄
├── 키우자 타운 팀                  → 키즈카페 교재만 봄
└── BBQ 로맨스 팀                   → BBQ 교재만 봄
```

**핵심 원리**:
- 교재는 **팀 단위**로 배포 (어느 지점이든 같은 팀이면 같은 교재)
- 지점은 **위치 구분용** (리더보드, 통계용)

## 폴더 구조

```
content/
├── 유치원/         (→ AION 킨더 팀에게 배포)
│   ├── _원본교재/
│   ├── _브랜드/
│   └── _결과물백업/
│
├── 키즈카페/       (→ 키우자 타운 팀에게 배포)
│   ├── _원본교재/
│   ├── _브랜드/
│   └── _결과물백업/
│
└── KBBQ/           (→ BBQ 로맨스 팀에게 배포)
    ├── _원본교재/
    ├── _브랜드/
    └── _결과물백업/
```

## 업로드 방법

### 1. `_원본교재/` 에 파일 넣기

**파일명 규칙**:
```
01_등원맞이.pdf       (첫 번째 챕터)
02_위생방역.pdf       (두 번째 챕터)
03_식사지도.docx      (세 번째 챕터, 형식 혼용 OK)
```

**지원 형식**: PDF, Word(.docx), 한글(.hwp), 텍스트, 이미지 스캔본

### 2. `_브랜드/` 에 (선택) 톤 가이드

**TONE.txt** 예시:
```
회사명: 키우자 베트남
톤: 따뜻하고 신뢰감 있는 전문가
대상: 베트남 현지 교사 (한국어/베트남어 혼용)
강조: 아이 안전 최우선
피할 것: 꼰대 표현, 단정적 명령조
```

### 3. 업로드 완료 후 말씀

```
"유치원 교재 업로드 완료"
또는
"키즈카페 교재 업로드 완료, 시작해줘"
```

## 제작 품질 기준

### 슬라이드 (챕터당 10~15장)
- 원본 교재 내용 그대로 반영
- 캐릭터 + 스토리텔링 (지루함 방지)
- 슬라이드당 40초 이내
- 이모지는 TTS 낭독 방해하니 본문에서 배제

### 이미지
- 실전 상황 → 사실적 다큐 사진 톤
- 이론/개념 → 깔끔한 인포그래픽
- 판단 상황 → 플로우차트

### 퀴즈 (챕터당 3문항)
- 시나리오 기반 (실제 상황 제시)
- 교재 정독해야만 맞는 난이도
- 난이도 분포: 쉬움 20% / 보통 50% / 어려움 30%
- 오답 해설 포함

### 교육 효과 장치
- 핵심 포인트 1문장 강조
- 법적 필수 사항 색상 강조
- 실수 방지 팁 별도 표시
- 체크리스트 형식 활용
- 챕터 끝에 요약 슬라이드

## 자동 배포 흐름

업로드 → 제작 → 배포까지 제가 자동 처리:

1. 원본 파일 파싱 (PDF → 텍스트)
2. 슬라이드 스크립트 생성 (챕터당 10~15장)
3. AI 이미지 생성 (각 슬라이드)
4. 나레이션 3언어 (한/영/베)
5. 퀴즈 설계 + 오답 해설
6. Supabase 업로드 (sop_documents)
7. Published 상태로 전환
8. 카테고리 자동 설정 (유치원/키즈카페/KBBQ)
9. 학습 경로 자동 매칭:
   - 카테고리 "유치원" → AION 킨더 팀에게 배포
   - 카테고리 "키즈카페" → 키우자 타운 팀
   - 카테고리 "KBBQ" → BBQ 로맨스 팀

## 확장성

**매장 추가 시**:
- 관리자 페이지 → 지점/팀 관리
- 새 지점명 추가 (예: "베트남 하노이점")
- 자동으로 3개 팀(킨더/타운/로맨스) 같이 생성 필요

**새 업종 추가 시** (예: 한식당):
- 새 팀 추가 (예: "한식 수라상 팀")
- `TEAM_TO_CATEGORY` 매핑 업데이트
- 새 카테고리로 교재 생성
